# Belief Space Mapping

This project explores religious and philosophical orientations as a low-dimensional space defined by:

- Belief (atheism ↔ theism)
- Certainty (agnostic ↔ gnostic)
- Mode (internal spirituality ↔ social/institutional religion)

We collect survey data and use dimensionality reduction and clustering to identify emergent structures.

## Goals
- Map belief systems as vectors
- Visualize clusters and transitions
- Explore whether traditional labels (atheist, theist, etc.) reflect actual structure

## Structure
- /data: survey responses
- /notebooks: analysis
- /src: helper functions
